# Civic Issue Reporting System

A **Java Full Stack** application built with **Spring Boot**, **React.js**, and **PostgreSQL** to enable citizens to report civic issues and allow admins to track and resolve them efficiently.

## 🌟 Features
- Citizens can submit issues with title, description, and location.
- Admin dashboard to view, update, and resolve issues.
- Real-time notifications when issues are resolved.
- Status tracking: OPEN → IN_PROGRESS → RESOLVED.
- Secure authentication for users and admins.

## 🛠️ Technology Stack
- **Backend:** Java Spring Boot, REST API
- **Frontend:** React.js
- **Database:** PostgreSQL
- **Authentication:** Spring Security-JWT 
- **Tools:** Postman, Git
